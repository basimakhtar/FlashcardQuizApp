Flashcard Quiz App


This Android app is a flashcard quiz game that helps users differentiate between real life hacks and myths.

The goal is to improve critical thinking and help users identify misleading online advice.

The features include a system where users can answer questions based on a true/false selection. Every question is met with feedback. The score of every user is also being tracked and then displayed at the end of the quiz

Built using:
- Kotlin
- Android Studio
- GitHub Version Control
- GitHub Actions (CI Build)
